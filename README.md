# Projeto Agenda

Uma agenda de contatos onde se pode:

    * Ver todos os contatos
    * Buscar contato
    * Adicionar contato
    * Editar contato
    * Excluir contato

Projeto básico e inicial em python seguindo as aulas do curso de Python: 'Python Profissional na Prática' da [Solyd](https://solyd.com.br)
